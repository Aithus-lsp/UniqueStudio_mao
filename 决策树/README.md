    之所以会写这个，源于我对决策树的错误理解．我后来认为，决策树是利用特征选择划分对原数据（训练数据）的再处理，及对特征加权，进而我得出结论，可以在对原数据的结果依据特征加权判断原理重赋值后，可忽略决策树索引(特征)的顺序性，以原顺序对未知数据判断．后来越觉不对．
